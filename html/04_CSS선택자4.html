문자열 속성 선택자

  속성 값의 문자열을 확인하여 스타일 적용하는 방식의 선택자.


  1) 선택자[속성명 ~= "특정값"] { CSS 코드; }
      -> 띄어쓰기로 구분되어있는 여러 속성 값이 작성된 속성 중
      속성 값이 특정 값을 단어로 포함하는 요소를 선택


  2) 선택자[속성명 |= "특정값"] { CSS 코드; }
      -> 속성 값이 특정 값을 단어로 포함하는 요소를 선택
      단, "-" 기호로 구분, "-" 앞의 내용이 동일해야 한다.


  3) 선택자[속성명 ^= "특정값"] { CSS 코드; }   
      (^ : 캐럿)
      -> 속성 값이 특정 값으로 시작하는 요소를 선택


  4) 선택자[속성명 $= "특정값"] { CSS 코드; }
      -> 속성 값이 특정 값으로 끝나는 요소를 선택


  5) 선택자[속성명 *= "특정값"] { CSS 코드; }
      -> 속성 값이 특정 값을 포함하는 요소를 선택
div1 : name="aaa bbb str-1" class="str-class"
div2 : name="str-2 aaa" class="str-class"
div3 : name="str" class="class-str"
div4 : name="aaa str-3" class="str-class2" 
    
        
/* ~= : 여러 속성 값 중 하나라도 일치하면 선택 */
div[name ~= 'aaa']{
  background-color: greenyellow;
}
/* name이 aaa로 정확하게 하나인게 없어서 적용되지 않음
div[name = 'aaa']{ background-color: greenyellow;} */


/* |= : "-"(하이픈) 기호 앞쪽 단어가 일치하면 선택 */
div[class |= 'str']{
  color: red;
  font-weight: bold;
}
/* 안됨
div[class = 'str']{  color: red;  font-weight: bold;} */


/* ^= : 특정 값으로 시작하면 선택 */
div[class ^='class']{
  font-style: italic;
  font-size: 23px;
}


/* $= : 특정 값으로 끝나면 선택 */
div[class $= '2']{
  border: 3px solid blue;
}


/* *= : 특정 값을 포함하면 선택 */
div[name *= '2']{
  border: 3px dotted magenta;
}
